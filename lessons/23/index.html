SEO Checlist

1. Title - назва сторінки

<title> Назва сторінки - Назва сайта</title>

2. Meta description - опис сторінки

<meta name="description" content="140-160 символів max" >

3. Наявність favicon

Favicon Generator - генерує іконку і код для її підключення.

PS: Перевіряємо шлях до іконки

4. Meta-теги для пошукового робота

<meta name="robots" content="all" />

<meta name="revisit-after" content="1 days" />

Мовні версії сайту. Вказуються для пошукового робота, якщо такі є.
Якщо мова сайту тільки одна - ця частина не вказується взагалі.

<link rel="alternative" hreflang="x-default" href="https:/...mySite.com.ua/">

<link rel="alternative" hreflang="uk" href="https:/...mySite.com.ua/">

<link rel="alternative" hreflang="en" href="https:/...mySite.com.en/">

5. OpenGraph meta-теги + twitter

Open Graph protocol - мета-теги сервісу Facebook. Для надання базової інформації про сайт за посиланням. Тут є багато пунктів для специфічних тематик, та є і БАЗОВІ:


<meta property="og:locale" content="uk_UA" />
<meta property="og:type" content="website" />
<meta property="og:site_name" content="Назва_сайту" />
<meta property="og:title" content="Назва_сторінки_(див.пункт_1)" />
<meta property="og:url" content="https:/посилання_на_цю_сторінку/" />
<meta property="og:image" content="https:/img_що має представляти цю сторінку" />


developers.facebook.com - Open Graph validator

Twitter має схожі мета-теги


<meta name="twitter:title" content="Назва_сторінки_(див.пункт_1)">
<meta name="twitter:description" content="Опис_сторінки_(див.пункт_2)">
<meta name="twitter:image" content="https:/img_що має представляти цю сторінку">

6. Файл robots.txt у корені сайта

Як створити файл robots.txt - звичайний текстовий файл у корені сайту


User-agent: *
Allow: /
Disallow: admin/
Sitemap: https://посилання_на_сторінку_сайтмапи
Host: site_domain.com(без http & /)


facebook.com/robots.txt - приклад

7. Карта сайта sitemap.xml

Sitemap generator - створення мапи сайту мовою XML
